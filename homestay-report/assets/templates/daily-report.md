# 民宿经营日报模板

> 本模板定义日报的标准结构。Skill 读取本地数据后按此模板格式化输出。

## 模板变量

| 变量 | 来源 | 说明 |
|------|------|------|
| `{{date}}` | 系统日期 | 报表日期 |
| `{{weekday}}` | 系统计算 | 星期几 |
| `{{totalRevenue}}` | revenue.json | 当日总营收 |
| `{{revenueChange}}` | 环比计算 | 营收环比变化 |
| `{{occupancyRate}}` | orders + rooms | 入住率 |
| `{{adr}}` | revenue / 已售间夜 | 平均房价 |
| `{{revpar}}` | revenue / 可售间夜 | 每间可售房收入 |
| `{{newOrders}}` | orders.json | 新增订单数 |
| `{{cancelOrders}}` | orders.json | 取消订单数 |
| `{{checkinCount}}` | orders.json | 当日入住客人数 |
| `{{checkoutCount}}` | orders.json | 当日退房客人数 |
| `{{channelData}}` | orders.json 按平台分组 | 渠道分布 |
| `{{alerts}}` | 异常检测逻辑 | 异常标记列表 |
| `{{dataSource}}` | 采集时间记录 | 数据来源声明 |

## 异常标记示例

当检测到异常时，替换"各项指标正常"为：

```markdown
━━━ 异常标记 ━━━
⚠️ 入住率较昨日下降22%（75% → 58%）
   可能原因：工作日正常波动 / 天气影响
   建议：关注明后日预订情况，入住率持续低于50%可考虑促销

🚨 未来3天入住率仅28%
   建议：考虑启动限时优惠或渠道推广
```
