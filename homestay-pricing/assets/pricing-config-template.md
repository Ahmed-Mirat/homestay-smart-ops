### 成本底线计算参考

```
成本底线 = 房租日均分摊 + 物业水电日均 + 保洁费/次 + 布草洗涤费/次 + 耗材 + 人工分摊 + 折旧

示例：
- 房租分摊：5000元/月 ÷ 30天 = 167元/天
- 物业水电：约30元/天
- 保洁：80元/次
- 布草洗涤：40元/次
- 耗材（洗漱/茶水）：15元/次
- 人工分摊：约50元/天
- 折旧（装修/家具）：约30元/天
= 合计约 412元/晚
```
