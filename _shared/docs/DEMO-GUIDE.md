## 演示路线（15 分钟）

### 第一站：定价智能（5 分钟）

**打开看板 HTML**，展示：

| 面板 | 演示要点 |
|------|---------|
| KPI 卡片 | ADR ¥955 / OCC 72% / RevPAR ¥688 / 竞品均价 — 一屏看清经营健康度 |
| 竞品价格对比柱状图 | 竹悦云庐 vs 云溪山居/竹海人家/山间雅舍，按房型对齐 |
| 调价建议表 | 系统基于 5 因子模型自动计算建议价（含置信度标注） |

- "不需要商家后台，消费者端就能采集竞品价格"

## 核心卖点（一句话版）

1. **零门槛**：8 项核心功能无需任何 OTA 商家后台权限
2. **全自动**：定时采集竞品、自动排班、自动日终、自动催缴
3. **透明化**：每条定价建议都附带完整决策依据链
4. **双模式**：民宿 OTA 运营 + 公寓长租管理，一套系统覆盖
5. **企业微信集成**：通知/日报/告警直接推送到微信群
