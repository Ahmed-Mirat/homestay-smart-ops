- **共享核心层 (`_shared/`)**：作为系统的中枢，负责环境自动初始化 (`auto-install.js`)、配置管理 (`config.json`)、数据持久化 (`data/*.json`) 以及通用工具（如企业微信通知 `notifier.js` 和任务调度）。
- **业务模块层**：采用模块化设计，`tcm-inventory/` 独立处理进销存逻辑（入库、扣减、效期预警），`tcm-reception/` 与 `homestay-guest/` 分别承载接待与客服知识库能力。
- **交互驱动架构**：基于 `SKILL.md` 定义的触发词与 Agent 行为规则，通过自然语言指令调用底层 Node.js 脚本 (`inventory.js`, `widget-data.js`) 实现业务闭环，无需传统 GUI 界面。
- **数据流**：所有业务数据（库存、会员、交易）统一收敛至 `_shared/data/` 目录下的 JSON 文件，通过 `config-writer.js` 和各类业务脚本进行原子化读写。
